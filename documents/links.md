## Model to use

You should use CLIP ViT-B/32 from HuggingFace.
While you are welcome to test other models, this is the one we request everyone to use. If you evaluate other models, you should list results in your final report (notebook).

You can find it at https://huggingface.co/openai/clip-vit-base-patch32.
